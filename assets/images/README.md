# Suggested visuals for the case study

Drop exports here so the gallery loads automatically (see `main.js` — images appear when the file exists).

| Suggested filename | What to export from Figma / research |
|--------------------|--------------------------------------|
| `hero-poster.webp` or `.jpg` | Hero moment: key journey screen, DSM cover, or workshop photo (wide ~1600×900). Optional; hero works without it. |
| `gallery-01-journey.webp` | Redacted journey map, service blueprint, or “before/after” flow for booking or check-in. |
| `gallery-02-systems.webp` | Premium theme / component page, variables, or Dev Mode handoff snippet (blur confidential UI). |
| `gallery-03-ops.webp` | DesignOps snapshot: review cadence, utilization dashboard (numbers redacted), or committee slide. |
| `gallery-04-research.webp` | Usability test still, airport study, or CSAT summary (faces blurred if needed). |

**Export tips**

- **WebP** or **JPEG** at ~1200px wide keeps the page fast.
- Use **Figma** → frame → Export → 2x PNG then convert to WebP, or export WebP if available.
- Redact PNRs, revenue, and internal-only labels before export.
